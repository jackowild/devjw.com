Changelog:

v0.08:
  • Added ability to specify default IP address when running from the command line.

v0.07:
  • Keyboard shortcuts:
    • Ctrl+P = Play
    • Ctrl+R = Record
    • Ctrl+(1-8) = Play speed 1-8
  • Changed loop button functionality so that it loops the current clip instead of all clips.
  • Window can be resized.

Requirements

  • A BlackMagic Design HyperDeck Studio or Studio Pro on firmware version 4.1 or later.
  • A Windows PC running XP or later.
  • Microsoft .NET Framework 3.5 or later.
